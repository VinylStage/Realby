## Getting Started

### Install Package

    npm install

### Run Server

    npm run dev

---

# Project Purpose

사용자가 자신의 관심사나 주제에 대해 다양한 토픽의 글을 작성 및 게시하고, 블로거와 유저 간 소통을 극대화한 범용적인 블로그 플랫폼.

## Usage Language & FrameWork

Backend - Python, Django Rest Framework

Frontend - React, Next.js
